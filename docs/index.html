Welcome to the CaaS Documentation Hub

Here, you'll find comprehensive documentation for the CaaS Project, offering all the necessary information to kickstart your journey with CaaS.

Getting Started in Milo

Begin your CaaS journey in Milo by creating an account on Milo's Sharepoint authoring site at adobe.sharepoint.com. This account grants you the necesary access to the CaaS authoring tools.

Getting Started in AEM

Embark on your CaaS journey in AEM by accessing an AEM instance where CaaS functionality has been enabled. For more information, visit the "Enable CaaS in AEM" resource.
