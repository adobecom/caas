Welcome to the CaaS Documentation Hub

This is the documentation hub for the CaaS Project. Here you will find most of the information you need to get started with CaaS.

Getting Started in Milo

To get started with the CaaS in Milo, you will need to have an account in Milos sharepoint authoring site. You can do this by visiting the adobe.sharepoint.com page. where you will be able to access the CaaS authoring tools.

Getting Started in AEM

To get started with the CaaS in AEM, you will need access to an AEM instance where CaaS has been enabled. You can do learn more by visiting the Enable Caas in AEM.
