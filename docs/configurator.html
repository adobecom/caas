Introduction to CaaS - Milo Configurator

The following provides a concise overview of the CaaS Milo Configurator. The configurator is a tool that allows you to quickly create and customize your collection in real time. Using the configurator, you can easily visualize your collection in real time, customize the look and feel, and configure the collection layout to meet your specific needs.

Once the collection is created, you can easily add it to any Milo page by simply copying and pasting the collection link.

Important: The best way to lear how to use the configurator is to try it out yourself. Start creating your own collection and experiment with the different options available.

Basics

The Basics tab in the Milo CaaS Configurator allows you to configure the basic settings for your collection. The following options are available in the Basics tab:

Collection Name Collection name is used only for the collection link pasted in Milo.
(Optional field)
Collection Title Collection titles are displayed above the collection. The collection titles can be localized using placeholders.
(Optional field)
Collection Title Level Choose the appropriate collection title heading level for the page's SEO structure.
The default value is <h3>.
Source Select the source for the collection content. If no Source is selected, the data
Results Per Page Upload an image for your collection.
Total Cards to Show Enter tags for your collection.
Auto detect country & lang Use this checkbox to detect country and language automatically.
Country Enter a target for your collection.
Language Enter analytics for your collection.

For a more detail list of metadata options, please visit the CaaS Wiki page.

UI

The UI tab in the Milo CaaS Configurator allows you to customize the look and feel of your collection.

The following options are available:

Show Card Borders Enable this option to show borders around each card in the collection.
Disable Card Banners Enable this option to hide the card banners in the collection.
Use Light Text Enable this option to use light text in the collection.
Use Overlay Links Enable this option to use overlay links in the collection. With this options seleted the entire card becmes clickable.
Show total card count at top Enable this option to show the total card count at the top of the collection.
Card Style Choose the card style for the collection. The available options are:
  • 1/2 Card
  • 3/4 Card
  • 1/2 Height Card
  • Full Card
  • Double Width Card
  • Product Card
  • Text Card
  • Custom Card
Card Accessibility Title Level Choose the appropriate card title heading level for the page's SEO structure.
The default value is <h3>.
Layout Choose the layout for the collection. The available options are:
  • Grid
  • Carousel
Layout Type Choose the layout type for the collection. The available options are:
  • Default
  • Compact
  • Full
Grid Gap (Gutter) Enter the grid gap (gutter) for the collection.
Theme Choose the theme for the collection. The available options are:
  • Default
  • Dark
  • Light
Details Text Specified the behavior of the details text in the collection. The available options are:
  • Default - Shows the test entered in the card Details field
  • Last Modified - Shows the last modification.
Collection Button Style Choose the button style for the collection. The available options are:
  • Default
  • Primary
  • Secondary
Load More Button Style Choose the load more button style for the collection. The available options are:
  • Default
  • Primary
  • Secondary
Custom Card HTML Enter custom HTML for the cards in the collection. To access the data from the JSON response, use the following format:
<div>
  <h1>{$card.title}</h1>
  <p>{$card.description}</p>
</div>
CTA Link Behavior Choose the CTA link behavior for the collection. The available options are:
  • Same Tab/Window
  • New Tab/Window

For a more detail list of metadata options, please visit the CaaS Wiki page.

Cards

This tab is to configure the cards in the collection.

Featured Cards Cards listed here, either by URL or UUID, will appear on TOP of the collection when it loads regeardless of the sorting setting.
Excluded Cards Cards listed here, either by URL or UUID, will NOT appear on the collection
Hidden CTAs Cards listed here, either by URL or UUID, will NOT have CTAs
Tags that should not have CTAs Cards with the select the TAGs will not have CTAs

Sort

This tab is to configure the sorting options for the collection.

Default Sort Order Defines which Sort option will be preselected by default
Enable Sort Popup Enables the Sort drop down menu on the right side above the cards.
Sort options to display: Defines the sorting options avaible in the Sort dropdown

Bookmarks

This tab is to enable and configure the card Bookmark functionality. Bookmarks allow users to mark cards, similar to favorites.

Show bookmark icon on cards Enables the display of a bookmark (heart) icon on the footer of each card
Only show bookmarked cards Displays only those that have been bookmarked.
Show Bookmarks Filter Displays the Bookmarks filter in the filter panel.
Icon Link for in 'My Bookmarks' URL for custom icon
Icon Link for not in 'My Bookmarks' URL for custom icon

Search

This tab is to enable and configure the collection Search functionality.

The location and behavior of the search input will depend on the filters position, left or top.

Show Search Enables the Search input
Choose What To Search Through Specifies the data fields to be included search

*Important: Filters must be enabled for the Search to apppear.
