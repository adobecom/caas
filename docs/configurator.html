Introduction to CaaS - Milo CaaS Configurator

The following provides a concise overview of the CaaS Milo Configurator tool. The cofigurator is a tool that allows you to create and customize your collection in real time. Once you have created your collection, you can simply copy and paste the collection link in your Milo page.
